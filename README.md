# Primetrade.ai_Project_Assignment

# Task Management Application

A full-stack Task Management Application built using Node.js, Express.js, MongoDB, React.js, and JWT Authentication.

This project was developed as part of the Primetrade.ai Backend Developer Internship Assignment.

---

# Features

## Backend Features

- User Registration
- User Login
- JWT Authentication
- Protected Routes
- CRUD APIs for Tasks
- Error Handling Middleware
- Password Hashing using bcryptjs
- MongoDB Database Integration
- RESTful API Structure

---

## Frontend Features

- Register User
- Login User
- Protected Dashboard
- Create Tasks
- Get Tasks
- Update Tasks
- Delete Tasks
- Axios API Integration

---

# Tech Stack

## Backend

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcryptjs
- cors
- cookie-parser

---

## Frontend

- React.js
- Vite
- Axios
- React Router DOM
